# 던전 앤 배추2 - 배추봇v2
## 소개
던전 앤 배추는 팀 크레센도에서 서비스하는 배추봇v2 기능입니다.  
자신의 펫을 만들어 여러가지를 즐길 수 있습니다.

__현재 이 기능은 개발중입니다.__
## 명령어
| 명령어  | 설명   |
|:-------|:-------|
|마이펫은?|마이펫의 현재 상태를 확인합니다.|
|* 쉬어|마이펫을 쉬게 합니다.|
|* 놀아|마이펫과 같이 놉니다.|
|* 탐험|마이펫을 던전에 보냅니다.|
|* 부화|마이펫을 생성합니다.|

기타 명령어 작업중..
